# 📦 Sustainable Packaging Recommendation System

A Python-based GUI application that recommends the best packaging types based on multiple weighted parameters such as cost, durability, environmental impact, and reusability.

The project uses **Tkinter** for the graphical interface, **Pandas** for data processing, and **Matplotlib** for interactive data visualization.

---

# 🚀 Features

- GUI-based Packaging Recommendation System
- Weighted score calculation
- Data normalization for fair comparison
- Interactive charts and visualizations
- Top packaging recommendations
- User-friendly interface

- Reusability

---

# ⚙️ How It Works

1. User provides weights for:
   - Cost
   - Durability
   - Environmental Impact
   - Reusability

2. Data is normalized.

3. A weighted score is calculated.

4. Packaging recommendations are generated.

5. Charts are displayed for better analysis.
